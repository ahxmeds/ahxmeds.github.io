# Personal Research Website

This is a static academic website that can be hosted directly with **GitHub Pages**.

## Files

- `index.html` — main page structure
- `styles.css` — styling
- `script.js` — mobile nav and publication filtering
- `assets/` — place your CV PDF, headshot, and any extra files here

## How to host from GitHub

### Option 1: User site (best)
Create a repository named:

`ahxmeds.github.io`

Then upload these files to the root of that repo and push to GitHub.
Your website will be served automatically at:

`https://ahxmeds.github.io`

### Option 2: Project site
Create any repo, for example:

`research-website`

Upload these files, then go to:

- **Settings**
- **Pages**
- Under **Build and deployment**, choose **Deploy from a branch**
- Select the `main` branch and `/root`

Your site will appear at a URL like:

`https://ahxmeds.github.io/research-website/`

## What you should customize

Open `index.html` and replace:

- email address
- Google Scholar link
- LinkedIn link
- paper titles, coauthors, and links
- CV file path
- any placeholder project text
